# Energy And Momentum

This folder contains [ONELAB](https://onelab.info/) simulation code for benchmarking open boundary electrostatic and magnetostatic simulations. The main quantities that are calculated are the energy and the electromagnetic momentum contained in the fields.

To run the simulations, open `main.pro` in Gmsh, select the problem type, and press the Run button. Other parameters are set directly in the simulation files, namely in `sphere_common.pro` and `main.pro`.

Subfolders contain simulations using other FEM software.
